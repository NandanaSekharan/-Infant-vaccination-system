# Infant Vaccination Management System

## 📌 Project Description
This project is a web-based application developed to manage infant vaccination records efficiently. It helps hospitals and parents to track vaccination schedules, search records, and generate vaccination certificates.

---

## 🚀 Features
- 👶 Infant registration system
- 🏥 Hospital search functionality
- 📍 Location-based search
- 📄 Vaccination certificate generation
- 🔍 Vaccination record search and view
- 👤 Parent dashboard for managing details

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP

---

## 📂 Project Structure
- `admin/` → Admin functionalities
- `parent/` → Parent module
- `database/` → Database files
- `css/`, `js/`, `images/` → Frontend assets
